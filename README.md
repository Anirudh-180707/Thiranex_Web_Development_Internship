# Thiranex Web Development Internship Projects

This repository contains all projects completed during the Thiranex Web Development Internship program.

## Overview

The internship focuses on developing practical web development skills through a series of progressively advanced projects, covering modern frontend technologies, accessibility standards, responsive design principles, JavaScript programming, API integration, and full-stack deployment.

## Task 1: HTML5 Semantic Structure & Accessibility

Developed a multi-page personal portfolio website using:

* Semantic HTML5 elements
* ARIA accessibility attributes
* SEO-friendly meta tags
* Accessible contact form
* WCAG accessibility guidelines

### Technologies Used

* HTML5
* Accessibility (WCAG)
* SEO Best Practices

## Task 2: Advanced CSS3 & Responsive Architecture

Enhanced the portfolio website using modern CSS techniques:

### Technologies Used
